App Goldfish
📖 Instrucciones
🎮 Objetivo

Controla a Goldfish y sobrevive el mayor tiempo posible esquivando obstáculos y recolectando burbujas para recuperar vidas.

🕹️ Controles

← → Mover izquierda / derecha
A D Alternativa izquierda / derecha
Mouse Mueve el pez siguiendo el cursor
Clic En una burbuja para reventarla

💎 Obstáculos

Rocas 🪨 y troncos 🪵 caen desde arriba. Evítalos o perderás una vida. Máximo 4 de 5 carriles ocupados simultáneamente.

🫧 Burbujas

Aparecen tras perder la 1ª y 2ª vida (una sola vez cada vez). ¡Tócalas con el pez o haz clic sobre ellas para recuperar una vida!

⚡ Dificultad

La velocidad y frecuencia de obstáculos aumentan progresivamente, aunque puede bajar aleatoriamente en intervalos largos para darte un respiro.

💀 Game Over

Al perder las 3 vidas el juego termina. Tu puntuación se compara con el récord.
